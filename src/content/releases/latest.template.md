# Latest Release - Template

This file is a template for the release note fragment that can be generated by CI from the ShaleGame repository.

Suggested fields:

- Title (e.g., "v0.1.0 - First Playable")
- Date
- Short summary
- Notable changes / bullet points
- Links to assets or downloadable builds

Place the generated markdown (or HTML) into `src/content/releases/latest.md` to have it appear on the site.
